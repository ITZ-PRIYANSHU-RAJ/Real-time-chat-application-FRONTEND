# Real-Time Chat Application – Frontend

## 📌 Overview

This repository contains the **frontend of the Real-Time Chat Application** built with **React.js**.
It provides a modern user interface where users can sign in, view chats, and send messages instantly.

The frontend communicates with the backend API to manage authentication, users, and messages.

---

## 🚀 Features

* User login and registration interface
* Real-time messaging UI
* Responsive design for desktop and mobile
* Chat interface with message history
* API integration with backend server

---

## 🛠️ Tech Stack

* React.js
* JavaScript
* HTML5
* CSS3
* Axios (API requests)
* Socket.io Client (real-time communication)

## ⚙️ Installation

### Clone the repository

```
git clone https://github.com/YOUR-USERNAME/chat-app-frontend.git
```

### Navigate to project folder

```
cd chat-app-frontend
```

### Install dependencies

```
npm install
```

### Start the application

```
npm start
```

The frontend will run on:

```
http://localhost:3000
```
